Vibrating String GUI

  This GUI demonstrates the behavior of a vibrating string with different initial conditions.

Three methods are employed to solve the wave equation of the vibrating string with the given initial condition. These methods are:

  • Finite Difference Scheme
  • Fourier Scheme
  • Newmark Scheme

The string is considered to have both ends fixed. The length of the string is given by L. It is in equilibrium along the x-axis as shown in Fig. 1. In order to simplify the calculations, the following assumptions are made:

  • There is no force due to gravity
  • The string has a uniform density (mass/unit length)
