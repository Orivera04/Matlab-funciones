gjgraphs.m

Program to demonstate how elementary matrix row operations as used in 
Gauss-Jordan elimination may be interpreted graphically. 


multr.m

Function for row operation on matrix that multiplies every element of one row 
by a constant.


radd.m

Function for row operation on matrix that adds one row to another row.


mradd.m

Function for row operation on matrix that adds constant multiple of one row to
another row.


rSwap.m

Function for row operation on matrix that interchanges two rows.
