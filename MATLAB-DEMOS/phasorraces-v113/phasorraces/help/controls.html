  New Question

Pressing this button will generate a new question for the user. The type of question depends on the setting in the 'Type of Test' pop-up menu box.

Back to Top
  Type of Test

This pop-up menu may be used to specify the type of question that the user wants to be tested on.

Choices include the following:

  • Complex   - Addition of Complex numbers in Phasor/Rectangular form.
  • Sinusoid   - Addition of Sinusiods.
  • RealPart   - Real part of complex exponential to cosine signal.
  • Spectrum  - Conversion to/from Complex exponential from/to Cosine forms.
  • Sampling  - Finding normalized radian frequency from sampling period.
  • ZTransfom  - Locating Pole/Zeros for given transfer function.
  • All      - Any one of the above types, choosen at random.

      Start Timer

    Pressing this button, displays a New Question and starts the TIMER. If this is pressed without pressing the New Question button, then the question remains the same, but the TIMER starts running from 0.

    Back to Top
      Stop Timer

    When the user is ready with the answer, pressing this button, stops the TIMER and the time taken to calculate the answer is shown next to the TIMER. Pressing this button while the TIMER is not running, has no effect.

    Back to Top
      Show Answer

    After having stopped the timer, pressing this button will display the solution for posed question. Pressing this button without pressing the Stop Timer (i.e while the TIMER is running) does not have any affect.

    Back to Top
      Alternate Answers

    This push-button will only appear if one of the options Sinusoid or RealPart is selected under the Type of Test popup menu, and after the TIMER has been stopped. Pressing this button shows results that differ in phase from the initial solution by multiples of 2p. Upto five different choices are shown, and after this the answers start repeating.

      Rectangular Form

    This checkbox will only appear if the Complex option is selected under the Type of Test popup menu. Checking this box will display the rectangular form of the Question and its solution. Unchecking this will hide the rectangular form and show the polar form.

    Options Menu: Level

    This menu item will reveal two options: Novice or Pro. Whichever option is checked will determine the difficulty level of the questions produced by pressing the New Question button.

    TIMER

    This text field displays the time taken by the user to answer a question. i.e the time elapsed in between pressing Start Timer and Stop Timer once.

    Main text fields

    The main screen of the GUI has three display fields, which dont have any Names identifying them, but are the main fields in the GUI.

  • Title section - This field shows the Type of Test with a brief description and is at the top of the main screen in the GUI.
  • Question section - This is where the question is displayed. This section lies to the left of the New Question and Type of Test.
  • Answer section - This is where the answer is displayed. In the case of ZTransform the Pole-Zero plot is displayed. This section lies to the left of the Show Answer button.
