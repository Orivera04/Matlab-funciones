Digital Karma Help Document

 

Digital Karma generates matrices & graphically represents them.  I created it to experiment with one & two dimensional cellular automata.  However, the calculation of matrices has no foreseeable limitations and is not limited to cellular automata specifically.  Digital Karma can apply a set of calculations or rules to each cell/value generating a new matrix(2-D).  Alternatively, the rules can be applied to only the last row pushing all preceding rows upward(1-D).  The preceding matrices or rows become scrollable.  This is fundamentally just plotting some rule set applied to a matrix & displaying as a grid of cells changing colors.  The main difference is values are denoted by color & not position as in plots.
