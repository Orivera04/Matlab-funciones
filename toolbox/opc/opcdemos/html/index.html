OPC Toolbox Tutorials

The OPC Toolbox extends the MATLAB technical computing environment to include functions for interacting with OPC Data Access Servers. The toolbox provides these main features:

  • An object-oriented framework for interacting with one or more OPC servers directly from MATLAB.
  • Compliance with the OPC Foundation Data Access Standard, version 2.05a.
  • Event-driven interaction with OPC servers.
  • The ability to log data from OPC Data Access servers without the need for Historical Data Access support on the OPC Server.
  • An intuitive Graphical User Interface for browsing OPC Servers, creating OPC Toolbox objects and initiating logging tasks.
