Labels and figure properties

  

 

 

Title

plt(...,'Title',t);
Inserts the title string t above the plot area.
Default: no title

 

LabelX

plt(...,'LabelX',s);
Uses the string s as the x-axis label.
Default: 'x axis'

 

LabelY

plt(...,'LabelY',s);
Uses the string s as the main (left hand) y-axis label.
Default: 'y axis'

 

LabelYR

plt(...,'LabelYR',s);
Uses the string s as the auxiliary (right hand) y-axis label.
Default: no label

 

FigName  

plt(...,'FigName',f);
Use the string f as the name for the plt figure window.
Default: plt

 

Position

plt(...,'Position',p);
Specify the figure size and position on the screen in pixels:
p = [x(left) y(bottom) height width]
Default: [9 9 700 525]
