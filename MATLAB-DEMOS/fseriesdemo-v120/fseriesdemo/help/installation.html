Installation

Installation: Extract the files from the archive, making sure to retain the original folder structure. If you wish, add the directory to the MATLABPATH.

To Start: If you have added the installation directory to the MATLABPATH when you installed it, you can just run the fseriesdemo.m function and the GUI will start. If you have not added the installation directory to the MATLABPATH, first change to the directory in which the files were extracted and then run the fseriesdemo.m function. The program will ask you if it is okay to add the installation directory to the path for the duration of the current MATLAB session. Answer yes.
