Embedded Target for OSEK/VDX Demonstrations

The Embedded Target for OSEK/VDX allows you to run Real-Time Workshop Embedded Coder generated code on a MPC555 based OSEK environment.

Core capabilities include:

  • Automatic build, download via SingleStep, and run on target hardware
  • Automatic build and launch of SingleStep debugger
  • Targeting of OSEKWorks and ProOSEK Implementations
  • OSEK Library for a subset of OSEK OS APIs and example hardware driver
  • Report generation with Oil file and RAM/ROM measurements
