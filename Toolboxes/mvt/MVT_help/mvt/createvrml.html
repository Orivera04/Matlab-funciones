createvrml

CREATEVRML Creates a VRML-file for use with the Visualization toolbox.

FLAG = CREATEVRML(DATA, WORLD, TARGETFILE) creates a VRML-file TARGETFILE based on the infomation in DATA (see VERIFYDATA for help on DATA). The VR- world is created from the VRML file WORLD, which must be located in the Visualization Toolbox subfolder 'world'.

If the field 'type' is not specified in DATA, 'type' is set equal to the first entry in type list.

See also TYPEDEF, VERIFYDATA, DIRECTOR.
