Simple LED Model

Simple model demonstrating use of device driver blocks provided with the Embedded Target for Motorola MPC555 real-time target.

How to open this demo:

Click the link below or at the command line type mpc555rt_led.

Right-click on the Target_LED block, and select Real-Time Workshop -> Build Subsystem. Then use the Download Control Panel to download the generated Target_LED_RAM.s19 file to your Phytec phyCORE-555 board and observe the blinking LEDs.

There is a detailed tutorial describing how to use this model to generate and download a stand-alone real-time application. See Tutorial: Creating a New Application in the Embedded Target for Motorola MPC555 User's Guide.
