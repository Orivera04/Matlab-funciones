Multi-tasking and task execution profiling

This model illustrates the multi-tasking scheduler and the task execution profiler. It has been configured to load the processor such that task pre-emption occurs. You can download and run this model, then run profile_c166 to upload execution profiling data and present the results.

The execution profiling shows the behavior when a slower timer-based task is pre-empted by a faster one.
  1. To open this model, click the link or at the command line type c166_multitasking
  2. To generate code and download to the target, press CTRL+B (or select Tools-> Real-Time Workshop->Build Subsystem). Click Build in the following dialog to continue the code generation process. Once downloaded, click Run in the debugger to start the application on the target.
  3. At the command line, type profile_c166.

See the Execution Profiling section in the User's Guide for a detailed explanation of this demo.
