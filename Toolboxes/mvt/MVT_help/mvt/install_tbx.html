Installing the Marine Visualization Toolbox

To use the Marine Visualization Toolbox with MATLAB make sure to perform the following steps:

  • Make sure the files MVT_toolbox.zip and MVTinstall.m are located in the same folder
  • Make this folder your current directory in MATLAB
  • Run the toolbox installation file and specify the input as follows:
          MVTinstall('toolbox')

You should now be able to run the MVT as any other toolbox. To verify successful installation, type help mvt.
