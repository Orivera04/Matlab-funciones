  Get x(t) | h(t)

These buttons bring up a new window which allow you to pick x(t) and h(t).  A list box at the top provides a limited choice of signals.  If one wishes, the signal's parameters (e.g., length, amplitude, etc...) can be changed with the controls along the right edge of the window.  When satisfied clicking the OK button imports the chosen signal into the main window.

Note:  The signal may not be fully displayed when changing the signal parameters.  To replot the signal, click inside the plot box.

  Flip x(t) | h(t)

These radio buttons allow you to change which signal is being flipped.  Flipping a signal means reflecting the signal to the other side of the origin.

Note:  The current time indicates the sliding location of the flipped signal.  For example, the default starting position is to plot the signal with the t =  -2.  Thus, whatever value of the flipped impulse response was at the origin is shifted to t = -2.

  Index Label:  t = #

There are two labels for the current time.  Dragging these with the mouse will change the value of the current time.  The current time can also be changed by using the numeric 4 and 6 keys.

  Plot Box:  Signal / Flipped Signal

This is a plot of two signals.  The first signal labeled Signal is either the Input signal or the Impulse Response depending on the settings of the Flip x(t) | h(t) radio buttons.   Accordingly, the signal being flipped is labeled Flipped Signal.  Changing the current time will change the location of Flipped Signal.

  Plot Box:  Multiplication

This plot shows the result of the multiplication of the two signals displayed in the Signal / Flipped Signal plot box.  The plot changes depending on the relative overlap between the two signals.

Back to Top
  Plot Box:  Output

This plot shows the output of the linear convolution between the Input signal and the Impulse Response.

Note that the value at the current time is highlighted in the Output plot.  The numeric value (i.e., the value on the y axis) of this sample is determined by integrating the area within the multiplication plot.
