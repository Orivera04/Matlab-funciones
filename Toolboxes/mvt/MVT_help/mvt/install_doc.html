Installing Help on the Marine Visualization Toolbox

To use help on the Marine Visualization Toolbox with the MATLAB help browser make sure to perform the following steps:

  • Make sure the files MVT_help.zip and MVTinstall.m are located in the same folder
  • Make this folder your current directory in MATLAB
  • Run the toolbox installation file and specify the input as follows:
          MVTinstall('help')

The MATLAB help browser needs to be restarted to become aware of the newly added help files, make sure to do this.

You should now be able display the MVT help in the MATLAB help browser. Type doc mvt/ to verify this, the help browser should now display the MVT roadmap.
