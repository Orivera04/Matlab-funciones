Fixed-point fuel control system

This demo model was derived from the demo fuelsys.mdl. The floating point control algorithm has been converted to fixed point in order to allow efficient code generation for the C166® microcontroller. Note that this demo model requires licenses for Fixed-Point Blockset, Stateflow and Stateflow Coder.

  1. To open this model, click the link or at the command line type c166_fuelsys
  2. To generate code and download to the target, right click on the controller subsytem and select Tools -> Real-Time Workshop ->Build Subsystem. Click Build in the following dialog to continue the code generation process.

See the Simple Example Applications Tutorial in the User's Guide for details on which settings in this demo reduce RAM and ROM requirements compared to the original model.
