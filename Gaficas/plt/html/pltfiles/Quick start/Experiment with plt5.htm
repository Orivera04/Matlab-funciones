Experiment with plt5.m

  

 

 

The easiest way to start learning about plt is to start the sample script plt5.m. Just type plt5 at the command prompt. This simple script plots 5 traces, the last of which is plotted on the right hand axis (although it is off initially). Experiment by trying the following:

  • Left and right click on all the objects in the plt window.
  • Click and drag on the x and y axes tick labels (try both left and right buttons)
  • Left or right click and drag in the plot area. (On and off the traces do different things).
  • Double click and drag in the plot area
  • Click on the text Line 5 to enable the last trace and the right-hand axis. Repeat above experiments.

Expand on what you have learned playing with plt5 by:

  • Using plt with some data you are currently working with. Type plt(x,y) just as you would with Matlab's plot routine.
  • Use the plt workspace plotter by typing plt with no arguments. Then you can interactively select the variables you wish to plot.
  • Run the other programming examples.
  • Explore the topics in "Using the plt window" from the plt help home page.
