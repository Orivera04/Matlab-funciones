C166 I/O Driver Library

The Embedded Target for Infineon C166® Microcontrollers includes a small I/O driver library that lets you add I/O driver blocks to your controller model for interfacing to hardware. The blocks are contained in the following sublibraries:

  • Asynchronous/Synchronous Serial Interface
  • CAN Interface
  • Execution Profiling
  • TwinCAN Interface
