  Plot Options:   Set Line Width

This menu item allows you to interactively change the default line width used in all the plots.  This feature was implemented as a lecture aid since thin lines are sometimes hard to see from the back of a room when projecting an image of a computer screen.

  Plot Options: Show All Plots

If this menu item is selected, plots for input signal, sampled signal and reconstructed output are all shown. If this is not set, the plot of reconstructed output is not shown.

Back to Top
  Plot Options: Show Radian Frequency

If this menu option is set, the frequency axis in the continuous domain spectrum plots are labeled in 'w'(rad/sec). If it is not set, then the frequency axis is in analog frequency 'f'(Hz).

Back to Top
  Plot Options: Show Discrete Radian Frequency

If this menu option is set, the frequency axis in the Discrete domain spectrum plot is labeled in 'w'(rad) and periodic over the range -p to p. If it is not set, then the frequency axis is in normalized cyclic frequency, and periodic over the range -0.5 to 0.5.

  Plot Options: Show Lower Frequency Signal

This menu item will toggle the display in the discrete domain signal, between a low and high frequency components of an aliased signal. It does not have any effect, when there is no aliasing.
