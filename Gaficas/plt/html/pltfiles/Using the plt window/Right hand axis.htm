Right-hand axis

  

Enabling

If either the LabelYR or YlimR arguments were specified, the last trace will be displayed on a separate y axis on the right hand side of the plot. This is sometimes referred to as the auxiliary axis and the associated trace as the auxiliary trace. If the last trace is disabled (from the trace selection box) then the auxiliary axis will also disappear.

Cursoring

The cursor controls for the auxiliary trace work a bit differently. The auxiliary trace can never be the active trace, although in some sense it is always active. The main cursor (displayed as a plus sign) may be moved to any trace except the auxiliary trace. At the same time that the main cursor is visible, the auxiliary cursor is also visible. It's shown as a small circle with the same horizontal position as the main cursor, but its vertical position is set to fall on the auxiliary trace. When the auxiliary axis is visible, the y-cursor edit box doubles up. The first edit box is for the active trace as usual, and the second edit box contains the y value for the auxiliary trace. The colors of these edit boxes are different (matching their respective traces) which makes it clear which trace applies to each cursor readout.

Panning and zooming

The following controls affect both the main and auxiliary axes simultaneously:

  • LinX/LogX (menu box)
  • LinY/LogY (menu box)
  • Panning the X-axis
  • Zooming the X-axis

However you can't pan or zoom both the left and right side Y axes at the same time. The following actions can change the left-hand y-axis limits but can NOT change the auxiliary axis limits:

  • Zooming with an expansion box
  • Typing in y-axis limits using the cursor edit boxes
  • Panning or zooming in the plot area

The only way to change the right-hand axis limits is to right or left click on or near one of the right-hand axis ticklabels and dragging them to the desired position. (As before left click/drag is for panning and right click/drag is for zooming).
