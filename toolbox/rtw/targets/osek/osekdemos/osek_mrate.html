OSEK Multirate/Multitasking Demo

This demo model for the Embedded Target for OSEK/VDX illustrates how multirate models are executed in a multitasking fashion using OSEK. Each rate in the model is mapped to an OSEK Task. Each Task is activated by an Alarm at appropriate rate using the System Counter. Inspection of the generated file "osek_main.c" and "<model>.oil" shows this mapping.
