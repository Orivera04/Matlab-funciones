Copy lgrid.ocx to $WINDIR\system32

Open a command shell, change to $WINDIR\system32 and type

	regsvr lgrid.ocx

$WINDIR is your windows directory.

For more information on LiteGrid please visit
	http://www.codeguru.com/control/LiteGrid.shtml

The compiled version lgrid.ocx is modified according to
Qi Yang to ensure correct highlighting.
