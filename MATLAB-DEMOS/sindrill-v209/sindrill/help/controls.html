  Your Guess:  Show Guess

Checking this box will superimpose a plot of the guessed sinusoid onto that of the quiz sinusoid in the Plot Box in addition to displaying its mathematical formula immediately above the box.  Unchecking it will remove the guessed plot and formula.

  New Quiz

Pressing this button will generate a new quiz sinusoid.

  Plot Box

This box displays the plot of the quiz sinusoid whose characteristics are to be guessed.   If the Show Guess box is checked, a plot of the guessed sinusoid entered by the user is displayed on top of the quiz sinusoid.

Back to Top
  Answers Menu

This menu displays the actual amplitude, frequency and phase of the quiz sinusoid.

Back to Top
  Options Menu:  Level

This menu item will reveal two options:  Novice or Pro.  Whichever option is checked will determine the difficulty level of the quizzes produced by pressing the New Quiz button.

Back to Top
  Options Menu:  Set Line Width

This menu item enables adjustment of the plotting line widths.  Use the sliding bar to change the current line width; the preview box above the sliding bar will update to show the currently selected width.  Pressing OK will close the window and apply the new line width to the gui.

  Help Menu:  Contents

Pressing this button will launch the host computer's internet browser and to load the index.html page found in the help directory.  If an error occurs you can manually load the page into your browser.
