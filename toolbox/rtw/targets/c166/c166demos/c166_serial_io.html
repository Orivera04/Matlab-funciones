C166 Serial Input/Output Demo

This demo model for the Embedded Target for Infineon C166® Microcontrollers provides a demonstration that illustrates use of the Serial Transmit and Serial Receive blocks.

  1. To open this model, click the link or at the command line type c166_serial_io
  2. To generate code and download to the target, press CTRL+B (or select Tools-> Real-Time Workshop->Build Subsystem). Click Build in the following dialog to continue the code generation process. You will be prompted to reset the target.

There is a tutorial explaining this example in detail in the User's Guide, see "Tutorial: Simple Example Applications for C166® Microcontrollers".
